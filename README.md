# Todo Application

A simple and clean **Todo List web application** built using **Spring Boot**, **Thymeleaf**, and **Bootstrap**.  
The app allows users to manage daily tasks with basic CRUD operations and a clear visual indication of task completion.

---

## 📌 Features

- Add new tasks
- View all tasks in a list
- Mark tasks as completed (toggle)
- Delete tasks
- Visual distinction for completed tasks (strikethrough)
- Server-side rendering using Thymeleaf
- Responsive UI using Bootstrap

---

## 🛠 Tech Stack

### Backend
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA

### Frontend
- Thymeleaf
- Bootstrap

### Build Tool
- Maven

### Database
- MySQL

---

## 📷 Application UI

The application provides a minimal and user-friendly interface:

- Input field to add a new task
- List of tasks displayed below
- Completed tasks are shown with **strikethrough styling**
- Action buttons:
  - **Toggle** – mark task as completed or active
  - **Delete** – remove task from the list
